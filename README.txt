◇ 概要 ◇
　これはブラウザ上で遊べるアクションRPGのゲームです。
　Internet Explorerでは動作しない可能性があります。

◇ 動作条件 ◇
　Google Chrome
　Fire Fox

◇ ファイル構成 ◇

◇ インストール ◇
　インストールは必要ありません。
　index.htmlでゲームが起動します。

◇ アンインストール ◇
　アンインストールは必要ありません。
　通常の削除で消去できます。

◇ 遊び方 ◇
　キャラクターの移動：アローキーで動きます
　　↑
　←↓→
　通常攻撃：Zキー
　
　キャラクターのライフは5つあり、ライフが0になってしまうとゲームオーバーとなってしまいます。
　敵に当たる、敵に攻撃される、罠にかかるなどのアクションによってライフが減ります。
　敵にもライフがありますので、敵によっては何回か攻撃する必要があります。
